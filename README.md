# Expense Tracker

A simple React expense tracker built with Vite. Uses context, reducer, and local storage. No CSS library used.

Created by following: [ðŸ”— Build an Expense Tracker | React Hooks & Context API | Traversy Media](https://youtu.be/XuFDcZABiDQ).

## How to Run

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start the development server: `npm run dev`.
